How to run project

run_me_analyse.py is sample of analysis transaction service. when you run this file some samples files will be read from inputs and their analysis results will be written in oututs directory. run_me_sort.py is sample of sorting transaction service. when you run this file some input files will be sorted in /sort_result directory. Files are categorised by transaction type.

Different Modules For analysing transactions

operations

Required operations to analyze transactions are in this module

readers

Different reading strategies for different file extensions are in this module.
read_file function is our public API which finds appropriate reader and return the result of reading.

services

Required services are in this module and is the entry point of applications for other sub systems.

writers

writing strategies for different file extensions are in this module.
write function is our public API which finds appropriate writer and write file.

Different modules for sorting transactions

For sortings transactions i insert all input transactions in mongodb. Then i create index on collections and then sort collections and export result to chunked files. On reading files and writing files application works in parallel.

config.py

Mongodb configuration is in this module. So please set its values with correspond to your mongodb configuration

importer.py

read_file is the main API of this module read_file get queue item which has path,file_name and file_extension. Then this function select an appropriate reader for reading file and import it in mongodb.

partitioner.py

Partitioner module is responsible to read transactions from repository and create chunked file for every transaction type. This modules uses different strategies to write different type of files.

repository.py

repository module is responsible for interaction with mongodb. Bulk Insertion
Creating indexes
Reading from collections
are operations that this module can do

service.py

sort_transactions of this module is the main entry point of our API
